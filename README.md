# My Portfolio Website

A personal website created with the vision to showcase my skills, built using Next.js and TypeScript, providing an easy way to explore my work and get in touch.

## About

This is my personal portfolio website where I showcase my skills, projects, and experience as a developer.  
If you’re interested in collaborating or have any work opportunities, feel free to contact me through the contact form on this site.  
I’m always open to exciting new challenges and connections!

Check out the live site here:

## Features

- Built with **Next.js** for server-side rendering and fast performance.
- Typed with **TypeScript** for improved developer experience and type safety.
- Contact form integrated with **Nodemailer** to send emails directly from the site.
- Responsive and mobile-friendly design.
- Easy to customize and extend.

---

## Technologies Used

- NextJs
- TypeScript
- Nodemailer
- Tailwind CSS
